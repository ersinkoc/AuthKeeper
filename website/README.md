# AuthKeeper Website

Modern documentation website for AuthKeeper.

## Features

- React + Vite + TypeScript
- shadcn/ui components with Tailwind CSS
- Dark/Light theme support
- JetBrains Mono font
- Fully responsive
- GitHub Pages deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Automatically deployed to https://authkeeper.oxog.dev via GitHub Actions.
